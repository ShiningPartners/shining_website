享曆服務簡介

享曆工業股份有限公司是台灣專業配電器材製造業製造商. 享曆自西元1978年起開始提供專業的端子台、電機端子台、端子台連接器、螺絲端子台、端子盤、接線座、配電端子台、保險絲座、保險絲管夾、固態繼電器、SSR、絕緣礙子、支持礙子、獨立端子、匯流排支持座、纜線夾。...等服務. 擁有超過35年的專業經驗, 享曆總是可以達成客戶各項品質要求.

TA系列 軌道組合式端子台

軌道組合式端子台 (600V, 10A, 20A, 30A, 40A, 60A, 100A, 150A, 200A, 300A, 400A)
▲ TA系列 35mm軌道 組合式端子台

享曆 TA系列 軌道組合式端子台包括 TA-010~TA-400。額定電壓: 600V。額定電流: 10A~400A。安裝於 35mm的軌道上。可組成任意P數。

享曆SHINING TA系列 35mm軌道式端子台

產品型號: TA-010 / TA-020 / TA-030 / TA-040 / TA-060 / TA-100 / TA-150 / TA-200 / TA-300 / TA-400

額定電壓: 600V 額定電流: 10A / 20A / 30A / 40A / 60A / 100A / 150A / 200A / 300A / 400A

安裝方式: 於 35mm軌道 / 面板 安裝 電極P數: 可組成任意P數

安規認證: RoHS

TA系列端子台為軌道式端子台，以黑色為顏色基本款式。產品顏色可根據客戶需求客製變更。本系列產品除了以軌道安裝，透過兩側的安裝孔，亦可直接安裝於面板上。電流額定規格有10A, 20A, 30A, 40A, 60A, 100A, 150A, 200A, 300A, 400A。產品配有PVC的透明防塵蓋(軟)或透明保護蓋(硬)，提高產品的耐用度。可自由組裝所需的電極P數。在碼標紙條方面，我們也提供印字服務。
